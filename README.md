# nifty-style

nifty-style是一个实用的样式库。

## 安装

从GitHub上下载[最新的nifty-style](https://github.com/ZHTGitHub/nifty-style)，或使用[npm](https://www.npmjs.com/package/nifty-style)安装:

```bash
npm install nifty-style
```

## 如何使用

```bash
import 'nifty-style'
```
